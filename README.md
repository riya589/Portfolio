# Portfolio

A modern, animated personal portfolio built with React and Vite.

## Tech Stack

- React 19
- Vite 8
- JavaScript (ES Modules)
- Framer Motion (UI animations and section transitions)
- Three.js + React Three Fiber + Drei (3D visuals and effects)
- CSS3 (custom component-level styling)
- ESLint 9 (linting and code quality)

## Features

- Smooth animated sections and reveal effects
- Interactive hero section with character eye tracking/parallax behavior
- 3D visual elements rendered with React Three Fiber
- Theme toggle with local storage persistence
- Responsive layout for desktop and mobile

## Getting Started

### Prerequisites

- Node.js 18+ (recommended: latest LTS)
- npm

### Installation

```bash
npm install
```

### Run in Development

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

### Lint

```bash
npm run lint
```
